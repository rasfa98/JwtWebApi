# JwtWebApi
Web API built with .NET 7 trying to use common best practices.

## Technologies used

- .NET 7
- JWT
- MySQL
- Dapper
- MailKit

## Features

- Swagger documentation
- Login/Register
- Forgot/Reset password
- Verify user
- Email support
